## Featured Projects — Case Studies

### NonstopGG — B2B Digital Gift Card Marketplace

**nonstopgg.com**

**Problem:**
The client needed a scalable B2B marketplace where retailers could browse, purchase, and instantly receive digital gift cards, game currencies, and entertainment subscriptions. The existing workflow was manual and couldn't handle growing order volume.

**Solution:**
Built the frontend with Next.js and TailwindCSS, implementing a product catalog with filtering, a secure multi-step checkout flow, and a retailer dashboard for order tracking. Integrated real-time inventory syncing and automated delivery confirmation. Collaborated with backend engineers on API design and payment gateway integration.

**Result:**
Platform launched on schedule with a catalog of 500+ digital products. Automated delivery cut order fulfillment time from hours to seconds. The system now serves hundreds of B2B retailers with zero manual intervention on standard orders.

**Tech:** Next.js, TypeScript, TailwindCSS, REST API

---

### First Aid Surgical — Medical Equipment E-Commerce

**firstaidsurgical.com/en**

**Problem:**
A medical equipment supplier needed an online presence to showcase their catalog of surgical instruments, hospital furniture, and medical supplies to international B2B buyers. They required multi-language support and a way for buyers to request quotes directly.

**Solution:**
Led the frontend development of a fully responsive e-commerce site with i18n (internationalization) support. Built a product catalog with category filtering, detailed product pages, and an integrated quote request system. Implemented SEO best practices and optimized Core Web Vitals for search visibility.

**Result:**
Delivered a production-ready multi-language platform that expanded the client's reach to English-speaking markets. The quote request system streamlined buyer communication, and SEO improvements brought organic traffic growth within the first quarter of launch.

**Tech:** Next.js, TypeScript, TailwindCSS, i18n

### Daily Bangladesh Time — English News Portal

**en.dailybangladeshtime.com**

**Problem:**
Bangladesh Time, originally a Bangla-only React.js news portal serving 5,000+ daily users, needed an English-language version to reach a broader audience. The original codebase was difficult to scale and lacked server-side rendering for SEO.

**Solution:**
Rebuilt the English version from the ground up using Next.js with server-side rendering for SEO performance. Implemented role-based access control for multi-user content management, optimized image loading and page transitions, and reduced server costs by 20% through efficient data fetching and caching strategies.

**Result:**
Launched a fast, SEO-friendly English news portal with role-based editorial workflows. Server-side rendering improved search engine indexing, and the optimized architecture handled traffic spikes without performance degradation.

**Tech:** Next.js, TypeScript, TailwindCSS, PostgreSQL, Prisma

---

### Mellow Bangla — React Native Radio App

**mellowbangla.com**

**Problem:**
Wanted to build a native radio streaming experience for Bangla music and talk stations that worked reliably on Android with background playback — something the existing web-based alternatives couldn't deliver smoothly.

**Solution:**
Built the app solo using React Native for the frontend and NestJS for the backend API. Implemented background audio playback, station management, and push notifications. Designed a clean, native Android UI with offline-capable elements.

**Result:**
Shipped a production-ready Android app with smooth background streaming, low latency station switching, and a responsive UI that works seamlessly on low-end devices.

**Tech:** React Native, NestJS, TypeScript
